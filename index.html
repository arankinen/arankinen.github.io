3D Inverse Kinematics Simulator for 3 DOF Robot Leg

The idea was to make a 3D Simulator with Three.js (WebGL) for developing movement patterns for a hexapod robot (see above). A path for one 3 DOF leg of the robot can be simulated with parabola, trapezoid, trigonometric or straight line functions and the joint angles are stored in an object.
